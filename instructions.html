Welcome to the Memory App! This is the home of the Memory Competition, and also a fun brain training website. Here you will find activities designed to push your brain to its limits, and make your memory better than ever.

How to play

The game contains 4 variables that you must commit to your short-term memory within a short time frame. These variables are: Names, Cards, Numbers, and Sentences. You will have a certain amount of time to memorize each set of these 4 variables, and then you will respond with the correct answers after a set amount of time (designed to make the game a bit more difficult). You will then receive your results after submitting, along with all of your responses.

Singleplayer Mode

This is the game mode that you will use most often. Singleplayer Mode will allow you to play the game as much as you want, without the need for anyone else to play with (or against). There are 4 options within singleplayer mode: Easy, Medium, Difficult, and Custom. The first three are presets determined by us here at the Memory App, but the last one gives you complete control over your experience. You can make the game as easy or as hard as you would like, and play it until you can beat your previous record.

Multiplayer Mode

This mode will primarily be used to hold Memory Competitions, but it can also be used by you to play with friends or family. Simply create a game via the tab, share the code with whoever you would like to play with, and then hit start once everyone has joined!

Note: You will not be able to play the game under the create tab, that is only used for supervision. If you would like to host and play simultaneously, then open up a new join tab, and enjoy!
